# Marketech Digital — Next.js build

## Run locally
npm install
npm run dev

Then open http://localhost:3000

## Deploy on Vercel
- Create a new Vercel project
- Upload or import this folder
- Framework preset: Next.js
- Build command: next build
- Output: default
